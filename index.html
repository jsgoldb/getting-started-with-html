What is the web?

The "web" is a collection of documents written in Hypertext Markup Language, or HTML. These documents are interpreted and displayed by browsers.

To find a file, a browser makes a request via the internet to a server, using a hypertext transfer protocol, or HTTP.

It's important to remember that computers are stupid , so programmers have to be extremely careful to use correct syntax at all times

More about HTML

HTML is made of:
-text content (what you see)
-markup (how the content is arranged)
-references to other documents
-links to other pages

HTML consists of elements, made up of tags.
Most elements have a opening tag and a closing tag, with
the contents in between, unless it is a void tag, which contains
no elements. Each tag in turn can have attributes, which are
always assigned a value.

Inline elements work in-line, like the br, or break, tag.
Block elements, like the p, or paragraph, tag, create an "invisible" box around the contents, which have dimension.
